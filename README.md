## 🔍 프로젝트 소개

- 회원가입을 할 수 있습니다.
- 로그인을 해야 할 일 관리를 할 수 있습니다.
- 로그아웃을 할 수 있습니다.
- 할 일을 입력할 수 있습니다.
- 할 일을 체크하여 완료 표시를 할 수 있습니다.
- 할 일을 삭제할 수 있습니다.
